Descripción del proyecto 
Este proyecto permite tomar una cadena de texto y modificarla de manera dinámica haciendo uso de un matriz definida para hacerlo no legible, mientras este encriptado, luego este mismo texto mediante un botón puede ser desencriptado para leer el texto original. 

Funcionalidades

-	Se debe ingresar el texto a encriptar en el tex área y se oprime el botón encriptar
-	En el panel del lado derecho aparecerá el texto encriptado 
-	Le damos copiar el texto encriptado con el botón copiar
-	Y lo pegamos en el tex área con la función nativa de Windows 
-	Y hacemos click en desencriptar y el texto nos debe aparecer al lado derecho.
